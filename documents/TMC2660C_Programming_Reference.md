# TMC2660C 编程参考手册

> 基于 TMC2660C Datasheet Rev. 1.03 (2022-FEB-15)

## 1. 概述

TMC2660C 是一款用于两相双极步进电机的高性能驱动芯片，集成功率 MOSFET，支持高达 4A 电机电流。

### 1.1 主要特性

| 特性 | 描述 |
|------|------|
| 电流能力 | 最高 4A 峰值电机电流 (2.8A RMS) |
| 工作电压 | 5V - 30V DC |
| 微步分辨率 | 最高 256 微步/整步 |
| 封装 | 10x10mm QFP-44 |
| 接口 | SPI + STEP/DIR |

### 1.2 核心技术

- **StallGuard2™**: 高精度无传感器负载检测
- **CoolStep™**: 负载自适应电流控制，节能高达 75%
- **SpreadCycle™**: 高精度斩波算法
- **MicroPlyer™**: 256 微步平滑插值 (1/16 步输入)

## 3. SPI 接口

### 3.1 SPI 配置

- **模式**: SPI MODE 3 (CPOL=1, CPHA=1)
- **数据位**: 20 位
- **字节序**: MSB 优先
- **最大时钟**: 内部时钟时 4MHz，外部 16MHz 时钟时 8MHz

### 3.2 SPI 时序

```
CSN  ─────┐                                           ┌─────
          └───────────────────────────────────────────┘
SCK  ──────┐   ┐   ┐   ┐               ┐   ┐   ┐   ┐──────
           └───┘───┘───┘─── ... ───────┘───┘───┘───┘
SDI  ──────< b19 >< b18 >< b17 >... < b1 >< b0 >──────
SDO  ──────< b19 >< b18 >< b17 >... < b1 >< b0 >──────
```

| 参数 | 符号 | 最小值 | 典型值 | 最大值 | 单位 |
|------|------|--------|--------|--------|------|
| CSN 到 SCK 建立时间 | tCC | 10 | - | - | ns |
| SCK 低电平时间 | tCL | tCLK+10 | - | - | ns |
| SCK 高电平时间 | tCH | tCLK+10 | - | - | ns |
| SDI 建立时间 | tDU | 10 | - | - | ns |
| SDI 保持时间 | tDH | 10 | - | - | ns |

## 4. 寄存器定义

### 4.1 寄存器地址映射

| 寄存器 | 地址位 [19:17] | 功能 |
|--------|----------------|------|
| DRVCTRL (SDOFF=0) | %00x | 驱动控制 (STEP/DIR 模式) |
| DRVCTRL (SDOFF=1) | %00x | 驱动控制 (SPI 模式) |
| CHOPCONF | %100 | 斩波器配置 |
| SMARTEN | %101 | CoolStep 配置 |
| SGCSCONF | %110 | StallGuard2 和电流设置 |
| DRVCONF | %111 | 驱动配置 |

### 4.2 DRVCTRL - 驱动控制寄存器 (STEP/DIR 模式, SDOFF=0)

**写入格式**: `0x0xxxx`

| 位 | 名称 | 描述 |
|----|------|------|
| 19:17 | - | 寄存器地址 = %000 |
| 9 | INTPOL | 步进插值使能: 0=禁用, 1=16倍插值到256微步 |
| 8 | DEDGE | 双边沿触发: 0=仅上升沿, 1=双边沿有效 |
| 3:0 | MRES | 微步分辨率 |

**MRES 微步设置**:

| MRES | 微步/90° | 描述 |
|------|----------|------|
| %0000 | 256 | 最高分辨率 |
| %0001 | 128 | |
| %0010 | 64 | |
| %0011 | 32 | |
| %0100 | 16 | |
| %0101 | 8 | |
| %0110 | 4 | |
| %0111 | 2 | 半步 |
| %1000 | 1 | 整步 |

### 4.3 CHOPCONF - 斩波器配置寄存器

**写入格式**: `0x8xxxx` 或 `0x9xxxx`

| 位 | 名称 | 描述 |
|----|------|------|
| 19:17 | - | 寄存器地址 = %100 |
| 16:15 | TBL | 消隐时间: %00=16, %01=24, %10=36, %11=54 系统时钟周期 |
| 14 | CHM | 斩波模式: 0=SpreadCycle, 1=恒定关断时间 |
| 13 | RNDTF | 随机关断时间: 0=固定, 1=随机调制 |
| 12:11 | HDEC | 迟滞递减间隔 (CHM=0) 或 快速衰减模式 (CHM=1) |
| 10:7 | HEND | 迟滞结束值 (CHM=0) 或 正弦波偏移 (CHM=1) |
| 6:4 | HSTRT | 迟滞起始值 (CHM=0) 或 快速衰减时间 (CHM=1) |
| 3:0 | TOFF | 关断时间/MOSFET 禁用: 0=驱动器禁用 |

**TOFF 设置**:
- 0: 驱动器禁用，所有桥断开
- 1-15: 慢衰减时间 = 24 + 32×TOFF 系统时钟周期

**TBL 消隐时间**:
| TBL | 系统时钟周期 | @16MHz |
|-----|--------------|--------|
| %00 | 16 | 1.0µs |
| %01 | 24 | 1.5µs |
| %10 | 36 | 2.25µs |
| %11 | 54 | 3.375µs |

### 4.4 SMARTEN - CoolStep 配置寄存器

**写入格式**: `0xAxxxx`

| 位 | 名称 | 描述 |
|----|------|------|
| 19:17 | - | 寄存器地址 = %101 |
| 15 | SEIMIN | 最小电流: 0=1/2 CS, 1=1/4 CS |
| 14:13 | SEDN | 电流递减速度: %00=32, %01=8, %10=2, %11=1 次测量/递减 |
| 11:8 | SEMAX | 上阈值偏移 (上阈值 = (SEMIN+SEMAX+1)×32) |
| 6:5 | SEUP | 电流增量: %00=1, %01=2, %10=4, %11=8 |
| 3:0 | SEMIN | 下阈值/禁用: 0=禁用 CoolStep, 1-15=阈值×32 |

### 4.5 SGCSCONF - StallGuard2 和电流设置寄存器

**写入格式**: `0xCxxxx` 或 `0xDxxxx`

| 位 | 名称 | 描述 |
|----|------|------|
| 19:17 | - | 寄存器地址 = %110 |
| 16 | SFILT | StallGuard2 滤波: 0=标准模式, 1=每4整步更新一次 |
| 14:8 | SGT | StallGuard2 阈值 (有符号, -64 到 +63) |
| 4:0 | CS | 电流比例 (0-31, 实际比例 = (CS+1)/32) |

### 4.6 DRVCONF - 驱动配置寄存器

**写入格式**: `0xExxxx`

| 位 | 名称 | 描述 |
|----|------|------|
| 19:17 | - | 寄存器地址 = %111 |
| 16 | TST | 测试模式 (正常操作 = 0) |
| 15:14 | SLPH | 高侧斜率控制 |
| 13:12 | SLPL | 低侧斜率控制 |
| 10 | DISS2G | 禁用短路保护: 0=启用, 1=禁用 |
| 9:8 | TS2G | 短路检测延时 |
| 7 | SDOFF | STEP/DIR 接口: 0=启用, 1=禁用(使用 SPI) |
| 6 | VSENSE | 采样电阻电压范围: 0=310mV, 1=165mV |
| 5:4 | RDSEL | 读取数据选择 |
| 3 | OTSENS | 过温阈值: 0=150°C, 1=136°C (仅 TMC2660C) |
| 2 | SHRTSENS | 短路检测灵敏度: 0=低, 1=高 (仅 TMC2660C) |
| 1 | EN_PFD | 被动快速衰减/欠压阈值 (仅 TMC2660C) |
| 0 | EN_S2VS | 短路到 VS 保护 (仅 TMC2660C) |

**SLPH/SLPL 斜率控制**:
| 值 | 描述 |
|----|------|
| %00 | 最小 |
| %01 | 最小+温度补偿 |
| %10 | 中等+温度补偿 |
| %11 | 最大 |

**TS2G 短路检测延时**:
| 值 | 延时 |
|----|------|
| %00 | 3.2µs |
| %01 | 1.6µs |
| %10 | 1.2µs |
| %11 | 0.8µs |

**RDSEL 读取数据选择**:
| RDSEL | 读取内容 |
|-------|----------|
| %00 | 微步位置 |
| %01 | StallGuard2 值 |
| %10 | StallGuard2 + CoolStep 电流 |
| %11 | 所有状态标志 (仅 TMC2660C) |

## 6. 电流设置

### 6.1 电流计算公式

**RMS 电机电流**:
```
I_RMS = (CS + 1) / 32 × V_FS / R_SENSE × 1/√2
```

**瞬时电机电流**:
```
I_MOT = CURRENT_A/B / 248 × (CS + 1) / 32 × V_FS / R_SENSE
```

其中:
- CS: 电流比例设置 (0-31)
- V_FS: 满量程采样电阻电压 (VSENSE=0: 310mV, VSENSE=1: 165mV)
- R_SENSE: 采样电阻值

### 6.2 采样电阻选择

| R_SENSE (Ω) | VSENSE=0 时 RMS 电流 (A) | VSENSE=1 时 RMS 电流 (A) |
|-------------|--------------------------|--------------------------|
| 0.33 | 0.7 | 0.35 |
| 0.27 | 0.8 | 0.43 |
| 0.22 | 1.0 | 0.53 |
| 0.15 | 1.5 | 0.8 |
| 0.12 | 1.8 | 1.0 |
| 0.10 | 2.2 | 1.2 |
| 0.075 | 2.8 | 1.5 |

### 6.3 采样电阻电压阈值

| 参数 | VSENSE | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|--------|------|
| 峰值阈值电压 (低灵敏度) | 0 | 310 | 323 | 340 | mV |
| 峰值阈值电压 (高灵敏度) | 1 | 155 | 173 | 190 | mV |

## 7. 初始化序列

### 7.1 推荐初始化代码

```c
// 典型初始化序列
void TMC2660_Init(void) {
    // 1. 斩波器配置 (SpreadCycle 模式)
    TMC2660_WriteRegister(0x901B4);  // CHOPCONF: SpreadCycle
    // 或使用恒定关断时间模式:
    // TMC2660_WriteRegister(0x94557);  // CHOPCONF: Constant toff

    // 2. 电流设置
    TMC2660_WriteRegister(0xD001F);  // SGCSCONF: CS=31 (最大电流)

    // 3. 驱动配置
    TMC2660_WriteRegister(0xE0010);  // DRVCONF: 低驱动强度, RDSEL=StallGuard2

    // 4. 微步设置
    TMC2660_WriteRegister(0x00000);  // DRVCTRL: 256 微步

    // 可选: 启用 CoolStep
    // TMC2660_WriteRegister(0xA8202);  // SMARTEN: CoolStep, 最小电流 1/4 CS
}
```

### 7.2 寄存器值解析

**CHOPCONF = 0x901B4**:
- 地址 = %100 (CHOPCONF)
- TBL = %10 (36 时钟周期消隐时间)
- CHM = 0 (SpreadCycle 模式)
- RNDTF = 0 (固定关断时间)
- HDEC = %00 (16 时钟周期迟滞递减)
- HEND = %0110 (迟滞结束值 = 3)
- HSTRT = %100 (迟滞起始值 = 5)
- TOFF = %0100 (关断时间 = 4)

**SGCSCONF = 0xD001F**:
- 地址 = %110 (SGCSCONF)
- SFILT = 0 (标准 StallGuard2 模式)
- SGT = 0 (StallGuard2 阈值)
- CS = 31 (最大电流比例)

**DRVCONF = 0xE0010**:
- 地址 = %111 (DRVCONF)
- SLPH = %00 (最小高侧斜率)
- SLPL = %00 (最小低侧斜率)
- DISS2G = 0 (启用短路保护)
- TS2G = %00 (3.2µs 短路检测延时)
- SDOFF = 0 (启用 STEP/DIR 接口)
- VSENSE = 0 (310mV 采样电阻电压)
- RDSEL = %01 (读取 StallGuard2 值)

## 9. StallGuard2 使用

### 9.1 参数设置

| 参数 | 范围 | 描述 |
|------|------|------|
| SGT | -64 到 +63 | 阈值 (负值增加灵敏度) |
| SFILT | 0/1 | 滤波模式 |
| SG | 0-1023 | 测量值 (0=最大负载) |

### 9.2 调参步骤

1. 以合理速度运行电机，监测 SG 值
2. 逐渐增加机械负载
3. 如果电机堵转前 SG 未达到 0，减小 SGT
4. 如果 SG 先到 0 而电机未堵转，增大 SGT
5. 最佳设置: 堵转前 SG 在 0-400 之间

### 9.3 限制

- 极低速度 (< 1 转/秒): 背 EMF 太低，测量不稳定
- 极高速度: 无法驱动满正弦电流，响应变差

## 10. CoolStep 使用

### 10.1 参数设置

| 参数 | 范围 | 描述 |
|------|------|------|
| SEMIN | 0-15 | 下阈值 (0=禁用 CoolStep) |
| SEMAX | 0-15 | 上阈值偏移 |
| SEUP | 0-3 | 电流增量步数 |
| SEDN | 0-3 | 递减所需测量次数 |
| SEIMIN | 0/1 | 最小电流限制 |

### 10.2 工作原理

- 上阈值 = (SEMIN + SEMAX + 1) × 32
- 下阈值 = SEMIN × 32
- SG < 下阈值: 增加电流
- SG ≥ 上阈值: 减少电流
- 电流范围: 1/4 CS (或 1/2 CS) 到 CS

## 11. SpreadCycle 斩波器参数

### 11.1 推荐设置

| 应用 | TOFF | TBL | HSTRT | HEND |
|------|------|-----|-------|------|
| 通用 | 3-5 | 2 | 4-5 | 0-3 |
| 高速 | 2-3 | 1-2 | 4 | 0 |
| 低噪声 | 4-5 | 2 | 5-6 | 3-5 |

### 11.2 斩波频率计算

```
t_OFF = (TOFF × 32 + 24) / f_CLK
f_CHOP ≈ 1 / (2 × t_OFF)  // 近似值，取决于负载
```

## 13. 应用注意事项

### 13.1 PCB 布局

1. 采样电阻靠近芯片放置，单独接地
2. 电源滤波电容靠近 VSA/VSB 引脚
3. 5VOUT 电容 (470nF) 靠近引脚
4. 使用大面积接地平面

### 13.2 静止电流降低

- 静止时应降低电流至 10%-75% 运行电流
- 可节省 50%-90% 能耗
- 使用 CS 参数或独立模式的 CSN 引脚

### 13.3 ESD 保护

- 电机输出可加 100pF-1nF 电容
- 可选 LC 滤波器和压敏电阻
- 避免热插拔电机

## 14. 寄存器快速参考

### 14.1 写入寄存器地址

| 地址 | 寄存器 | 基础值 |
|------|--------|--------|
| %000 | DRVCTRL (SDOFF=0) | 0x00000 |
| %000 | DRVCTRL (SDOFF=1) | 0x00000 |
| %100 | CHOPCONF | 0x80000 |
| %101 | SMARTEN | 0xA0000 |
| %110 | SGCSCONF | 0xC0000 |
| %111 | DRVCONF | 0xE0000 |

### 14.2 常用配置示例

```c
// 256 微步, STEP 插值
#define DRVCTRL_256_INTPOL  0x00200

// SpreadCycle, TOFF=4, TBL=36, HSTRT=5, HEND=3
#define CHOPCONF_SPREAD     0x901B4

// 恒定关断时间模式
#define CHOPCONF_CONST      0x94557

// CoolStep 启用, 最小电流 1/4 CS
#define SMARTEN_COOLSTEP    0xA8202

// 电流设置 CS=31, SGT=0
#define SGCSCONF_MAX        0xD001F

// 驱动配置, RDSEL=01, SDOFF=0
#define DRVCONF_DEFAULT     0xE0010

// TMC2660C 推荐: 启用所有保护
#define DRVCONF_C_PROTECT   0xE001F
```

## 15. 与 TMC2660 (非 C 型) 差异

| 项目 | TMC2660C | TMC2660 |
|------|----------|---------|
| 静音性 | 更安静 | 标准 |
